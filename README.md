#### MODULE NAME
       OneBitAdder

#### DEPENDENCIES
       None

#### INPUTS
Name | Size(Bits)
-----|------------
Ci  |     1      
A   |     1      
B   |     1      

#### OUTPUTS
Name | Size(Bits)
-----|------------
Co  |     1      
S   |     1      

#### DESCRIPTION
       Computes  Ci  +  A  + B.  Stores two-bit result in Co and S;  Co is the
       first bit and S is the second.
